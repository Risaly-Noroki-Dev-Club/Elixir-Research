# Course Tracker

## Recording A Medication Event

1. Select the administration time.
2. Choose `Oral` or `Injection`.
3. Confirm dose, formulation, and optional note.
4. Review the second-check dialog before writing the log.

## Editing And Deleting

- `Edit` loads an existing record back into the editor.
- `Delete` removes one log after confirmation.
- Every add, edit, or delete operation recalculates the PK preview.

## Quick Add

- Quick add uses a saved long-term template for the current drug.
- You can override time, dose, interval, or note for a single log.
- The quick-add flow still ends in the same confirmation dialog.

## Route Guidance

- Oral guidance focuses on prescription confirmation, formulation handling, and safe recording boundaries.
- Injection guidance is a safety checklist only and does not teach injection technique.

## Metrics

- Course total: total dose accumulated for the selected drug.
- Last 24h: exposure from the previous 24 hours.
- Dose count: number of local log entries.
- Adherence: behavior summary, not a medical outcome.
